# 叉烧
---

| Ingredient | Quantity |
| ----------- | ----------- |
| 肉 | 2-3 lbs |
| 叉烧讲 | 1 瓶 |
| 蜂蜜 | 0.5 叉烧酱的重量 |
| 南乳 | 0.25 叉烧酱的重量 |
| 玫瑰酒 | 少许/20 g |
| 酱油耗油卤料碎片 | 适量 |



### 步骤
1. 酱料搅拌均匀
    - 玫瑰酒用来稀释，搅拌至浓稠
    - 酱油蚝油卤料用来调味，酱料可以稍咸
    - 保留少许酱料
2. 腌制1-3天
3. 烤：500F，15-20分钟，中途翻面涂酱




### 结果
- 表面caramalized， 内部入味
- 肉有嚼劲而不老
