# 日式咖喱
---

| Ingredient | Quantity |
| ----------- | ----------- |
| 牛肉flank | 随意 |
| 土豆等 | 随意 |
| 洋葱 | 半个 |
| 咖喱块 | 一个 |
| 生抽等 |  |
|  |  |



### 步骤
1. 洋葱炒至焦，加入锅中，加水烧开
2. 牛肉炒至表面焦，黑，入水煮开，撇去浮沫
3. 开始计时
4. 40:00：土豆萝卜等炒上色，入水
5. 50:00；加入咖喱块，调味
6. 55:00: 加青椒





### 结果
总体入味，口感好，酱油偏多
