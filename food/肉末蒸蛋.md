# 肉末蒸蛋
---

| Ingredient | Quantity |
| ----------- | ----------- |
| 肉末 | 随意 |
| 鸡蛋 | 3个 |
| 红葱头 | 3颗 |
| 葱蒜 | 随意 |
| 盐糖酱油 |  |



### 步骤
1. 將3顆蛋放入碗中，打勻，加入1:2或1:3的水、鹽、醬油
2. 用濾網濾出氣泡與沒打散的蛋液，放入滾水或電鍋中蒸約10分鐘
3. 蓋鍋蓋前，兩邊架筷子，讓鍋蓋不會完全密合，可減少蒸蛋出現的洞洞，保持形狀好看
4. 豬肉末入鍋，依序加入紅蔥頭、蔥煸炒
5. 肉末香氣出來後再放入蒜末，等肉熟，加入少許糖拌炒
6. 加入醬油、米酒、水之後，關小火煨煮，也可依喜好適量加入白胡椒
7. 取出蒸蛋，鋪上肉末，完成





### 结果
水掠夺略多
